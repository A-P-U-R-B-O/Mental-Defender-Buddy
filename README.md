# Mental Defender Buddy

**Mental Defender Buddy** is your AI-powered support system designed to help users recognize and defend against harmful thought patterns. It’s like having a rational friend in your pocket—without the judgment or the snack-stealing.

## What It Does

This app analyzes user input and responds with helpful, supportive advice. It's built to be emotionally intelligent and can help users reframe negative thoughts, reduce stress, and feel mentally supported through text-based conversations.

## Features

- AI-driven emotional support
- Friendly and non-judgmental interaction
- Customizable response logic (tweak it to your heart’s content)
- Lightweight and easy to run
